# Skill: UpdateConfig

When asked to update configuration, use the following rules:
1. Locate the config file (usually `claw.toml` or `.claw/config.toml`).
2. Read the existing contents first.
3. Apply changes preserving comments and structure.
4. Verify the syntax using the internal config parser via the notebook tool.
